2 例子

2.1 对序列的每一段应用一个函数

2.1.1 R

# generate data
a <- c(0:50)

# run tapply, find the max value for every segment
res = tapply(a, c(0:50)%/%10, FUN=max)
print(res)

0  1  2  3  4  5 
9 19 29 39 49 50

2.1.2 Python

a = range(0, 50)
res = [max(a[i:i+10]) for i in range(0, 50, 10)]

print(res)

[9, 19, 29, 39, 49]
