转换

集合<=>数组

  • double[] => List<Double>: 用guava的 Doubles.asList();
  • Set<Integer> => int[]: 用guava 的 Ints.toArray();

Stream <=> String

  • Stream转换为String: 使用Apache common/Guava库的IOUtils (可以不带第二个参数)
String myString = IOUtils.toString(myInputStream, "UTF-8");
  • String 转换为 Stream:
new ByteArrayInputStream(str.getBytes())

  • new HashSet： 使用 guava 的 Sets.newHashSet("a","b",...)  [参考链接]

 

 

JAVA 8 Stream  

过滤集合中满足条件的某个元素，不存在时返回空

myCollection
.stream()
.filter(prop -> prop.name.equals(name))
.findFirst()
.orElse(null);

 其中，最后一句也可换成orElseException ，则不存在时会抛出异常
